# Omara Release Manifests

This file (or future TOML/JSON) will define which versions of the components make up each Omara release.

Example:

## 2026.06 "First Light"

- omara-configs: v0.1
- omara-cli: v0.1
- omara-apps: v0.1
- omara-art: v0.1
- omara-os: v0.1
- omara-rpms: v0.1

This is the single source of truth for reproducible Omara images.
